# Images Folder

## Required Images

Please add the following images to this folder:

### 1. hero-background.jpg
- **Source**: First page of the CREATIVE INFRATECH BROCHURE.pdf
- **Description**: Clean industrial/steel structure image (no company logo)
- **Requirements**:
  - High resolution (1920x1080 or larger recommended)
  - Clean industrial/steel structure background
  - No text or company logo in the image
  - Professional, corporate appearance

### 2. logo.png (Optional)
- **Description**: Company logo for header
- **Requirements**:
  - Transparent background (PNG format)
  - Height: ~50px (will be auto-scaled)
  - If not available, the text "CREATIVE INFRATECH" will be displayed instead

## Notes
- The hero background image is required for the website to display properly
- If the hero background image is missing, a gradient fallback will be used
- The logo is optional - text will be shown if logo image is not available
